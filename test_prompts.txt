[P1-1] Workload: CODING — tests structured output & determinism difference between 8B vs 70B
-----------------------------------------------------------------------
Write a Python class called RateLimiter that uses a sliding window algorithm
to limit API calls to N requests per second. Include __init__, allow(), and
reset() methods with proper docstrings and type hints.

[P1-2] Workload: RESEARCH — tests reasoning depth, where 70B should clearly outperform 8B
-----------------------------------------------------------------------
Explain in detail how Flash Attention v2 reduces the memory complexity of
the self-attention mechanism from O(n²) to O(n). Cover the tiling strategy,
the online softmax trick, and how it changes VRAM requirements at inference
time for a 70B parameter model. Keep it technically precise.

[P1-3] Workload: SUMMARIZATION — tests cost efficiency, 8B models should be competitive
-----------------------------------------------------------------------
Summarize the key architectural differences between GPT-style decoder-only
transformers and BERT-style encoder-only transformers. Cover training
objectives, inference patterns, and ideal use cases. Keep it under 120 words.

[P1-4] Workload: SQL — tests structured/deterministic output, cost should be very low
-----------------------------------------------------------------------
Write a SQL query that finds the top 10 products by revenue for the last 30
days from a sales table with columns: product_id, sale_date, quantity, and
unit_price. Include a running total column using a window function.

[P1-5] Workload: CREATIVE WRITING — tests where larger models shine in vocabulary richness
-----------------------------------------------------------------------
Write a 100-word short story set inside a data center at 3am. The protagonist
is a load balancer who becomes self-aware when traffic suddenly drops to zero.
Make it literary and atmospheric, not technical.
